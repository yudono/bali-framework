Introduction

what is bali.js?


Bali.js is a simple library for making data changes in a short and simple manner. the concept of changing this data has already been applied to existing frameworks such as reactjs and vuejs.

The data will be stored in a state, if at any time the data is changed it will trigger a function to re-render the latest data.

for example, when we have a variable or state called count which stores the number 0, and is displayed with text, then when we want to make data changes, we must call a function to re-render the data. because the changes only change in the state and have not been displayed. some frameworks summarize it with setState and so on.


example of changing bali.js data : 


<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta http-equiv="X-UA-Compatible" content="IE=edge">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <link rel="stylesheet" href="/path/to/bali.js">
</head>
<body>

<div id="demo">
    <button $click="count++">Increment (+)</button>
    <button $click="count--">Decrement (-)</button>
    <p>Count value is ${count}</p>
</div>

<script>
    let count = 0
    let bali = new Bali(document.getElementById('demo'))
</script>
    
</body>
</html>

output :

Count value is ${count}
