how to use it?


You can use it as a trigger event listener, such as onclick, onchange, onkeyup and so on. when an event occurs, changes will occur in the state or related variables and attributes.


example : 


<div id="demo" class="mt-4 bg-gray-100 h-64 flex flex-col items-center justify-center">
    <input type="text" $keyup="name = $.value" placeholder="Your Name 1">
    <p class="py-4">My Name is ${name}</p>
</div>

<script>
    let name = ''
    let bali = new Bali(document.querySelector('#demo'))
</script>

output :

My Name is ${name}
